# Wallet Pytest Project

Simple wallet application implemented in Python and tested with pytest.

## Installation

Clone the repository and install dependencies (pytest):

```bash
git clone https://github.com/<your-username>/wallet_pytest.git
cd wallet_pytest
pip install -r requirements.txt  # or: pip install pytest
```

## Running the application

```bash
python wallet.py
```

## Running tests

```bash
pytest -v
```
